== Description ==

Replaces bbPress default textarea with the much more advanced CKEditor.


== Installation ==

###Installing The Plugin###

Extract the ZIP file to  `/wp-content/plugins/`.

Then just visit your admin area and activate the plugin.

Or use the plugin installer by navigating to "Plugins" -> "Add new" page and search for "Visual Editor For BBPress". Then click install and then activate buttons for related plugin.

**See Also:** ["Installing Plugins" article on the WP Codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins)

###Using The Plugin###

* Select version of CKEditor to use from "Settings" -> "Visual Editor For BBPress"
